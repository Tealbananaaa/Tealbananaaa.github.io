《班门：广场》笔记

# 《班门：广场》笔记

# 意大利的城市与广场

  • 佛罗伦萨劳伦齐阿纳图书馆 - 米开朗琪罗。室内大台阶应用的开端，楼梯开始成为室内装饰构件（那这样讲我们的舞美是对巴洛克早期美学的回应吗哈哈哈，太棒了，我喜欢大台阶果然是有理论基础的的）。

  • 柱子 - 装饰作用有时超过结构构件，手法主义。这和《建筑中国》中的说法相印证，西方古典建筑的雕饰大于了结构。正因如此，20cn 沙利文提出 “形式服从功能” 才具有如此的颠覆性吧。

  • 关于古迹保护：“孤零零地放在一片广场中央供着，完全破坏古迹和环境的原有关系，歪曲历史信息。”

    “中世纪的空气在这里有趣地凝聚着。”

    “我们古建保护着就是要教历史停止。” 古城的破败。旧城 “半死不活” 状态 - 现实的繁荣和古迹保护确实有矛盾。

  • 环境氛围 - 锡耶纳广场 高大建筑的影子挪移也是有趣的景观 形成日晷的效果。

  • “封建主蛮勇残酷的战争给这个小城留下的高昂激越的英雄故事，天主教会愚昧狡狯的迷信给这个小城留下的欢乐多彩的节庆活动。” 浪漫气息源于此。

    大为有理…… 节庆仪式源于愚昧。果真是长处的附近往往就藏着短处.

    “历史常常是不可深究的，那些令人陶醉的东西许多是从泪泉血海中孕育出来的，只是时间给他洗刷了腥气和涩味，反倒成为浸透乡土情谊的风尚习俗和文化传统了。”

# 经典城市广场与大师

  • 特拉法尔加广场。伦敦市中心交通便利处。小小的水面缓解枯燥空旷感。和那个大草坪一个意思。

    高差处修了个大台阶，刚好丰富空间层次。

  • 法国巴黎协和广场 - 城市中轴线上。

    八个角八座雕像界定界限。虚界面，又是你。

    大革命时期的刑场，和后面要说的嵇康有异曲同工之妙… 广场经常是见证血腥，罪孽或者战争的存在。

  • 梵蒂冈圣彼得广场

    热知识：梵蒂冈和天安门广场一样大。（……）

    贝尔尼尼设计。以圣彼得大教堂为核心，凸显它的核心地位，又要有聚拢人民的作用，有遮风避雨处。

    中间有一座方尖碑，如果直接以其为圆心则过于大而空旷。

    于是修建大小适中的圆，中间空的部分用一个方形来连接。同时圆形做了手脚，改为 “有方向性” 的椭圆，短轴在中轴线上让人们能离大教堂更近。回廊有 “母亲的拥抱感”。

  • 纽约时代广场

    聚合在商业营销的影响之下。阿道夫奥克斯，新闻人，时报广场。策划了最早的跨年狂欢。

# 环合与互峙：中国传统广场及其营造特点

只有古代文本记载中的遗存，特点与今日理解的广场大不相同。

  • 东汉张衡《西京赋》“临迥望之广场，程角抵之妙戏。”—— 看相扑比赛的场所

  • “麦收几处广场平，风清一缕炊烟直。”—— 乡村田野。对仗修辞，平与直，复合的空间感。

  • “云从月边度，影在地上行。”—— 完全的个人情趣，清幽，毫无公众性。

  • 苏州虎丘的千人石

    也具有公共活动属性，但零散很多。

    “天作广场堪共坐，人同明月岂无思。” 这个意境真的太喜欢啦！很多人一起坐在一片平整石地上看着月亮，竟然有种小池塘的感觉。

    古今 “广场” 根本不是一个概念。

    我们的广场，更接近院落。

  • “广” 作为动词而非形容词时的意义 —— 空间的弹性，人在场所中的感受和能动性。

    被景物衬托出的平整感，空旷感……

  • 唐 段安节：“永新乃撩鬓举袂，直奏曼声，至是广场寂寂，若无一人。”

    “从前人物空自哗，一顾广场皆寂寞。”—— 视觉状态的拥挤和听觉的 “空”。 有意思。

  • 营造特征 —— 知觉对比

    一些特别的视听体验，在一个广场上并置或复合 。两个或更多元素的对应搭配，互相衬托。

    “驰道尽，而有广场，中为月牙池，然得更广之。”

    上面的麦田。

    横向 - 纵向；广场和远树平整 - 自由形态；

    “广场杂瓜荒，古墙悬薜荔。” 画面感极强。

  • 主体的作用。某些仪式行为（比如祭拜）让人自发聚集。

  • 一个有趣的说法：“发现传统空间意象中隐藏的广场性，比如院，座，台等”。关注人的行为，什么样的空间都可能在一定时候体现出广场性。

# 罗马喷泉 —— 艺术、权力与自然的交响诗

很喜欢这个标题。关注了喷泉这一常见事物背后多个维度的内涵。

济慈长眠于罗马，留下墓志铭：

Here lies one whose name was write in water.

“此地长眠者，声名水上书。”

  • 破船喷泉贝尼尼。创意甚有趣，很戏谑。把喷水这件事不露痕迹地融入到场景里去，毫不穿凿。
  • 特里同喷泉、特莱维喷泉和四河喷泉，彰显教权。

  • 艺术家和水顺势而为的合作。
  • 基督教首都地位的确认：需要在城市空间中建立包括道路和指示物在内的朝圣视觉体系。两座大殿为朝圣者标识起终点，沿途喷泉强化线路的合理性。

# 换喻的风景：东大门设计广场

扎哈在韩国东大门的设计作品，这是古老的交易市场，历史与现代的交织。写的稍微有点官方，只有一段让我觉得有点启发性：

  • 借景造物。广场环绕古城墙和遗迹展开，建筑表面的空隙和褶皱空间提供了一览创新世界的窗口，在每一个不同的角度都可以 “窥探” 到不一样的历史遗迹或生活场景。

    我的理解是做成 “透” 的空间，让人可以穿过去直接看到后面原有的古迹，也就是用建筑构图，把原有历史建筑当作景观。

  • 三位建筑信息模型 BIM. 好像昨天还有人跟我说起。

# 凝动音乐的演奏

这一篇文字很少，也比较抽象。探讨摄影、建筑和音乐之间的关系，建筑师大都喜爱古典音乐，建筑又往往依托摄影呈现。前二者都是与光线的合作，后两者是对节奏的把握。

“建筑是凝固的音乐”→“建筑是凝动的音乐”（梁思成语）

“别的艺术都是让人感知理念，音乐则跳过理念，不依赖现象世界，是全部意志的直接客体化。”

“音乐是无影无踪，音乐是人活着的时候唯一可以感到的灵魂的一个部分。”

将艺术作品简化后呈现出的抽象结构作为一种纯粹形式…… 艺术本身隐藏着结构。作品本身诉说的精神内涵总是超出作品本身的表象。

宇宙飞船一样的透光感，妙的。

# 广场蒙太奇

以爱情电影《艳阳天》引入，评价圣马可广场是人类广场空间的理想范本。一个上演悲欢故事的舞台。不好意思我没看过。但是听简介，那种意难平但并不撕心裂肺的离别，还有点喜欢。

这个广场是一个公众性非常强的地方，承载世界各地游客。

  • 城市街区广场的日常使用率是周边建筑的多样性保证的。

    种类丰富的建筑承载种类丰富的人，维护空间全时段的活力。单一与强目的性是不利于活跃度的。

  • 关于广场的各个评价维度，甚是有趣：

    • 女人对坐在哪的选择会比男人更挑剔，对烦恼更敏感。如果一个广场里女性高于平均数那么说明广场不错。哈哈哈信哉此言。不过挑座位这件事是一位男老师教我的。

    • 阳光充足，有树，不在风口……

  • “存在主义” 的广场。

    • 最初为功能性而设，之后渐渐将商业与手工业驱逐。“恰恰在这个时期中，广场上出现了雕像、神庙和纪念建筑物。” 如此大众的场所，却是精神性极强的。是世俗的教堂。

    • 共性所趋，摒弃孤独的内在驱动。人往人处走。广场的强社交性。

    • 座椅 —— 飞机等，单向。广场则往往是无方向的，外向性，开放，社会活动最大可能。

    • “一个好的广场应该成为城市的喘息和留白”。 信哉此言。

    • 三角效应：相遇和交流的媒介，刺激人们联系的外部因素。比如雕塑、演出 etc。“话题”。 照此观点，广场针对的病症是孤独。

    • 用哲学发展解释广场的现象则是：

      • 柏拉图说现象世界之上存在着理念世界，于是广场方案图纸先于广场出现了；

      • 叔本华：人生虚无荒谬而孤独。尼采：上帝已死，超人要给虚无创造意义，于是我们建造广场。

      • 叔本华：审美行为可以消解虚无 —— 人们在广场上欣赏美。

      • 萨特：自由选择，积极行动；加缪：反抗孤独，反抗虚无 —— 人们开始交流。

      虽然很扯，但竟然也形成闭环，怪有趣的。

      想读萨特！！

随记：方与圆

圆形建筑：原始，不稳定，移动，无等级，不再扩张。(游牧民族建筑)

方形：成熟，稳定发展，等级序列，便于扩展边界。（成熟建筑）
