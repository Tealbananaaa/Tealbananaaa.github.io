# 不专业的中国建筑笔记

2023.09.01 开始读《图说中国建筑》。

原本是去逸夫馆寻找在诚品书店遇到的《半片砖瓦》，结果偶然刷到这本很不错的书。

原本晚上打算开个 word 写笔记，拖延了一下没动笔，结果就接触了写作工具 typora。

好吧，对建筑我是小白，对软件我也是小白。我一穷二白，白手起家。也没什么，瞎学吧。

# 1. 建筑要素

# 1. 造型布局

  • 基本的三段式造型：屋顶 屋身 台基（例：故宫 三重须弥座台基）
  • 中国建筑多长边方向开门而西方多短边方向
  • 多围合 内向型 封闭
  • 长边称面阔 短边称进深 两根立柱间称开间

# 2. 大木作

  • 大木作 木制结构构件，起承重作用
  • 小木作 门窗 家具
  • 术语体系：元代以前《营造法式》 明清时期《工程做法》
  • 柱 单棵树木主干制成 (传递载荷至地面)
  • 梁 进深方向（竖着）的水平承力 多为矩形 南方有一些圆梁 秀丽轻巧 称为月梁
  • 额 枋 檩 面阔方向（横着）的水平承力 大的是额小的是枋 最上面直接拖着屋顶的是檩
  • 椽 飞 檩上面盖的一层木条是椽 架住屋顶 飞是椽延长出来扬起的部分 通常唐代以后才有

# 3. 结构类型

风格差异大而结构类型差异小 （换外观不变内核）

  • 抬梁 最主流 柱顶置梁 梁端接檩 梁上放短柱 一层层往上。可以实现大跨度，少柱，开敞
  • 穿斗 常见于南方 柱密集 柱贯通整个空间 直接架檩 中间小梁串联
  • 井干 木材直接平行向上垒 极其费木 东北西南林区有见

【推测：北方政治和祭祀功能的大型建筑较多 南方多一些民居 木材也丰富】

【感慨一下。去年夏天实践在皖南乡下见到了很多穿斗式的屋子，文欣姐给我讲了这两种结构的大概。回来又在北京故宫观察了抬梁。今年冬天在广西的路上江枫给我指过一所井干式的屋子，可惜当时也没好好看清。想来有趣。谢谢这些可爱建筑人拉我入门。】

  • 石砌 夯土（台基） 砖拱券仿木

# 4. 斗拱

层层出跳，托举住上面的檐。装饰性强。通常用于高级建筑。明清时结构功能已经比较弱。

宋称为一朵斗拱，清称为一攒。

分为柱头、柱间、转角。（宋代叫 " 铺作”）

规格以拱与昂的出跳数量定义

# 5. 榫卯

凸出为榫 凹入为卯

（文字根本写不清 太过精妙）

# 6. 屋顶

造型最体现等级秩序，是一种标识物。影响单体造型和群组布局。

  • 庑殿顶 也叫 四阿顶 四个坡面。最高等级。【五台山佛光寺，后续依旧关注晋中建筑】
  • 悬山顶 两坡 屋顶两端悬挑 在山墙之外 遮风避雨保护木结构和墙 。最多见。常用于民居。
  • 歇山顶 悬山顶基础上加了一圈副阶 九脊 后期出现卷棚歇山 没有正脊 弧形屋顶
  • 硬山顶 悬山顶不出挑。砖石结构的山墙，不需要遮蔽保护。民居常见。
  • 攒尖顶 较小 园林常见 宫殿坛庙亦有。

一套严格细致的等级秩序 排序是重檐 > 单檐 庑殿 > 歇山 > 攒尖 > 悬山 > 硬山

观察故宫 正殿用庑殿，配殿差一点，前导再差一点。太和殿重檐庑殿 中和殿辅助 单檐攒尖 保和殿重檐歇山

# 7. 台基

功能：预潮。

常见夯土，外包砖石。方形吸取佛座造型变为须弥座。石砌。故宫台基均为汉白玉。

# 8. 小木作

  • 版门即对开的大门 宅院大门
  • 隔扇门 & 窗 小型建筑本体 上部棂花和下部裙板组成
  • 外檐木雕 在立柱上做大量雕刻装饰 【晋商大院、徽商民居 有钱的人雕梁画栋，后续仍旧关注】
  • 藻井 & 天花 藻井是穹顶 天花是水平方向 最初真的绘画藻类和水生生物，为了避火。
  • 彩画 在建筑结构上用颜色画图案就是彩画 比如额上。

# 10. 脊饰

象征主义文化传统的体现。鸱（一种海兽）的尾巴，与藻井类似，辟邪防火。

南北朝成熟，盛唐下方出现龙首形状，辽宋西夏明显龙头鱼尾，信仰的混合过渡阶段特征

金元明清完全变成龙形，名为 “鸱吻” 。

# 2. 时代风格

这部分开始综合单词进行造句和成篇。分析整体的建筑了。

作者依据时间线来讲述的，好处是时代背景分明，坏处是不能清晰地看出各类别建筑的沿革线索。于是记笔记的时候把同类建筑整理到一起。

鉴赏这种东西还是不能靠理论套的，没见过的建筑，描述得再厉害也形不成自己的理解。希望能有机会行万里路，亲眼多看一些古建吧。

其实看的也不算少。但，看实体的时候看不懂，看书的时候想不出，真是搞笑。

其实是没搞懂究竟在看什么 —— 关注作者讲述时的分析维度吧。

# 原始 - 城市、宫室

  • 氏族聚落
  • 统治中心与经济中心重合【延续至宋？）】
  • 夯土高台。【这其实挺有意思的，先垒一个高高的基座把自己抬升起来。之后也可以关注一下西方建筑有没有类似的操作】鼎盛：阿房宫

# 汉代 - 长安洛阳

  • 汉承秦制。长乐、未央（本意为欢乐无尽）【王南老师的《建筑史话》可以再复习一下】

# 三国两晋南北朝，南渡 - 城市与宫室

    1. 曹魏邺城（河北邯郸） 、洛阳
    2. 西晋洛阳
    3. 东晋、南朝建康【果然南方比较自由商业吗，开创坊巷，打破棋盘格，在内城保留】
    4. 北魏孝文帝再次迁都洛阳。

    【我在学历史吗】

# 早期 - 祭祀、坛庙

  • 祭祀大概可以分为两个维度：1 自然神 2 祖先；前者 “坛”，后者 “庙”。文化背景

# 秦汉 - 帝王陵、贵族墓

  • 覆斗形封土，形似金字塔 ——“倒斗” 可能就是这么来的吧。

# 原始、秦汉 - 居住建筑与园林

  • 从材料上，北方土木混合，南方纯木 材料
  • 所有园林的一大统一思路：以小见大，在小范围模拟一个大环境。【模拟天象、仙界或者是自然界。有意思啊。像个实验室……】
  • 汉代：坞壁。特殊时代背景特殊功能

# 魏晋 - 宅园

  • 玄学盛行 崇尚自然→向内的精神气质→ 园林致力于再现山水风光，静，隐逸。
  • 两晋贵族的 “别业”

# 隋唐 - 宅园

“有高度文化内涵的特殊人工环境”

  • 除了私家园林，还有公用的官府营建的园林，士大夫可以租用来欢宴庆祝

    【这个好有意思啊。古代士大夫就是有调调，比起今天在酒店租个宴会厅办婚礼什么的，租园林显然更有意趣……】

# 元明清 - 民居

  1. 明代：严密的住宅制度–民间不能用斗拱！不能用彩画！ 官员不能重檐！不能歇山！

  2. 清代毕竟经济发展，各地违反规矩多起来。

    • 北京四合院不得不乖乖听话 所以从不可能有斗拱
    • 晋商有钱且天高皇帝远，大量彩画，大量多重斗拱
    • 江南：院落小，进深大，高耸天井能遮阳，素雅，粉墙黛瓦木雕，额上好像不画那么多彩画

    地域、气候带来的特征

# 原始、秦汉 - 造型和装饰

  • 商周，建筑无留存，参考器物装饰，推测黑红为主 / 色彩 纹样
  • 汉代，脊部装饰，斗拱突出 构件（脊部、斗拱样式……）

# 唐代 - 造型装饰

中唐被视作分界点。前：素雅，灰与朱、白 后逐渐奢靡

丝绸之路 - 釉面陶、琉璃瓦 材料工艺 彩画 表面装饰

# 汉、魏晋 - 早期佛寺佛塔

  1. 东汉传入，三国以后动荡战乱，佛教大盛；
  2. 魏晋南北朝政权大都极力推崇佛教 政治、社会背景
  • 佛塔：模仿印度本土→本土化 发展沿革，本土化演变 这个最有意思了
    1. 楼阁式
      • 木结构、砖木结构、砖石结构仿木
      • 可以爬 各层距离统一而比较高
    2. 密檐式
      • 几乎都是砖石结构
      • 不能爬 内部是空筒 第一层很高而上面极其密
      • 楼阁式发展而来 算是一种功能性的消解？【变成密檐式的目的究竟是偶像功能加强还是变成了景观？】

# 隋唐 - 佛塔本土化、成熟

  • 舍弃了 “土”，改为砖木、纯木。
  • 崇拜属性消解 宗教意义逐渐淡化 变成景观

  1. 西安大雁塔

    方塔，砖仿木，纯装饰构件【这是什么一种思想根源，不是这种材料，强行模拟这种材料的结构，但也是杰作】常见的样式

  2. 山西五台佛光寺 祖师塔

    “墓塔”，大师的埋骨处。这个功能倒是鲜明。功能

  • 唐早期 -** 方塔 **/ 中唐以后 - 八边形塔更多 形态

  1. 山西五台南禅寺
    • 【唐代寺庙真的好好看好简洁大气，颜色也古雅 全中国只剩晋中留存 什么时候有机会看看…】
  2. 山西五台佛光寺
  3. 山西芮城广仁王庙
    • 道教建筑。但好像和佛寺区别不大…？
  4. 山西平顺 天台庵

皆是单檐歇山顶，不知道算这个时期的特点？还是这种功能的建筑的等级？

【为什么是晋中？为何山西有这么多佛教建筑？因为是北魏都城 or 李唐王室出生地？有待研究】

  1. 北宋崇道。

  2. 辽金西夏佛事极盛。

      善化寺。寺院建筑群，看上去好像是庑殿顶，好高规格。大木作 - 屋顶

# 宋辽金西夏 - 佛塔

  1. 两宋多见楼阁式（纯汉族风格），偶像功能弱。）纯木→砖

  2. 辽金多见密檐式（少数民族），相对原始的崇拜对象信仰特征。【懂了。是一种回头路】文化
