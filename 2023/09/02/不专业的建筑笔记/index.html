# 不专业的中国建筑笔记

# Part1. 《图说中国建筑》

2023.09.01 开始读。

# 1. 建筑要素

# 1. 造型布局

  • 基本的三段式造型：屋顶 屋身 台基（例：故宫 三重须弥座台基）
  • 中国建筑多长边方向开门而西方多短边方向
  • 多围合 内向型 封闭
  • 长边称面阔 短边称进深 两根立柱间称开间

# 2. 大木作

  • 大木作 木制结构构件，起承重作用
  • 小木作 门窗 家具
  • 术语体系：元代以前《营造法式》 明清时期《工程做法》
  • 柱 单棵树木主干制成 (传递载荷至地面)
  • 梁 进深方向（竖着）的水平承力 多为矩形 南方有一些圆梁 秀丽轻巧 称为月梁
  • 额 枋 檩 面阔方向（横着）的水平承力 大的是额小的是枋 最上面直接拖着屋顶的是檩
  • 椽 飞 檩上面盖的一层木条是椽 架住屋顶 飞是椽延长出来扬起的部分 通常唐代以后才有

# 3. 结构类型

风格差异大而结构类型差异小 （换外观不变内核）

  • 抬梁 最主流 柱顶置梁 梁端接檩 梁上放短柱 一层层往上。可以实现大跨度，少柱，开敞
  • 穿斗 常见于南方 柱密集 柱贯通整个空间 直接架檩 中间小梁串联
  • 井干 木材直接平行向上垒 极其费木 东北西南林区有见

【推测：北方政治和祭祀功能的大型建筑较多 南方多一些民居 木材也丰富】

【感慨一下。去年夏天实践在皖南乡下见到了很多穿斗式的屋子，文欣姐给我讲了这两种结构的大概。回来又在北京故宫观察了抬梁。今年冬天在广西的路上江枫给我指过一所井干式的屋子，可惜当时也没好好看清。想来有趣。谢谢这些可爱建筑人拉我入门。】

  • 石砌 夯土（台基） 砖拱券仿木

露明 指暴露室内顶部木构架，如果露明，就会对屋顶的构件桐油涂抹。对应的是用天花遮起来。

# 4. 斗拱

层层出跳，托举住上面的檐。装饰性强。通常用于高级建筑。明清时结构功能已经比较弱。

宋称为一朵斗拱，清称为一攒。

分为柱头、柱间、转角。（宋代叫 " 铺作”）

规格以拱与昂的出跳数量定义

# 5. 榫卯

凸出为榫 凹入为卯

（文字根本写不清 太过精妙）

# 6. 屋顶

造型最体现等级秩序，是一种标识物。影响单体造型和群组布局。

  • 庑殿顶 也叫 四阿顶 四个坡面。最高等级。【五台山佛光寺，后续依旧关注晋中建筑】
  • 悬山顶 两坡 屋顶两端悬挑 在山墙之外 遮风避雨保护木结构和墙 。最多见。常用于民居。
  • 歇山顶 悬山顶基础上加了一圈副阶 九脊 后期出现卷棚歇山 没有正脊 弧形屋顶
  • 硬山顶 悬山顶不出挑。砖石结构的山墙，不需要遮蔽保护。民居常见。
  • 攒尖顶 较小 园林常见 宫殿坛庙亦有。

一套严格细致的等级秩序 排序是重檐 > 单檐 庑殿 > 歇山 > 攒尖 > 悬山 > 硬山

观察故宫 正殿用庑殿，配殿差一点，前导再差一点。太和殿重檐庑殿 中和殿辅助 单檐攒尖 保和殿重檐歇山

# 7. 台基

功能：预潮。

常见夯土，外包砖石。方形吸取佛座造型变为须弥座。石砌。故宫台基均为汉白玉。

# 8. 小木作

  • 版门即对开的大门 宅院大门
  • 隔扇门 & 窗 小型建筑本体 上部棂花和下部裙板组成
  • 外檐木雕 在立柱上做大量雕刻装饰 【晋商大院、徽商民居 有钱的人雕梁画栋，后续仍旧关注】
  • 藻井 & 天花 藻井是穹顶 天花是水平方向 最初真的绘画藻类和水生生物，为了避火。
  • 彩画 在建筑结构上用颜色画图案就是彩画 比如额上。

# 10. 脊饰

象征主义文化传统的体现。鸱（一种海兽）的尾巴，与藻井类似，辟邪防火。

南北朝成熟，盛唐下方出现龙首形状，辽宋西夏明显龙头鱼尾，信仰的混合过渡阶段特征

金元明清完全变成龙形，名为 “鸱吻” 。

# 2. 时代风格

这部分开始综合单词进行造句和成篇。分析整体的建筑了。

作者依据时间线来讲述的，好处是时代背景分明，坏处是不能清晰地看出各类别建筑的沿革线索。于是记笔记的时候把同类建筑整理到一起。

鉴赏这种东西还是不能靠理论套的，没见过的建筑，描述得再厉害也形不成自己的理解。希望能有机会行万里路，亲眼多看一些古建吧。

其实看的也不算少。但，看实体的时候看不懂，看书的时候想不出，真是搞笑。

其实是没搞懂究竟在看什么 —— 关注作者讲述时的分析维度吧。

# Part2.《建筑中国 —— 半片砖瓦到十里楼台》

2023.09.04. 算是进阶阅读？其实这才是我想要读中国古建书籍的原点。上面第一本书做基本词汇扫盲，也很好。

# 亭台楼阁 - 建筑门类

# 城

  • “市井” 这个词源于井田，而非水井。井本身就是用来交换东西的地方。
  • 农业社会中 “城市” 事消费不事生产，经济上依赖乡村，所以说是政治军事的巨人和经济的侏儒。
  • 将城市的布局规划提升为 “大地文化”，有趣。方正的布局，“冷峻的大地文化”。
  • 中轴线 - 折半思想 - 文化脊梁 - 温馨而又严厉的气质。富于 “理趣”

THOUGHTS 有意思…… 从上一本书我已经记住了大概的术语，从这本书起更努力去体会建筑人对建筑的 “评价”。为何称某物美？美在哪？背后的成因、逻辑？和此前知识体系能否建立联系？是否可以迁移？

# 居

  • 四合院。坐北朝南为上，由于四合的结构，导致了每个位置屋子的地位差异。

    南首的屋子虽然南，但朝北开门开窗，叫做 “倒座”

    最北边的坐北朝南，但地位也不高。

    依据八卦的八个方位属性 —— 所谓风水。东 男性。西 女性。东北方是艮为，“鬼门”，非常阴。但古人的自我解救和小聪明艮为设厨房，明火照亮阴暗。 好有意思。

  • 苏州民居。“进” 和 “落” 结合，厅堂和天井庭院的穿插。

    与水融合。

    庭院为厅堂的延续，厅于庭交融。这个挺妙的，看上去阻隔了但在意味上还是连续的。

# 寺

  • 本土化 - 南北纵向中轴线，和民居宫殿等殊无二致。
  • 基本特点：力求南向、中轴对称，几重进深。
  • 山门 - 天王殿 - 院落 - 大雄宝殿。
  • 五台山 - 文殊道场 - 被开发为佛教圣地始于北魏孝文帝。

以对南禅寺的描述为例，观察一栋建筑的风格的角度：

“柱径很大”“柱身粗壮”“斗拱巨大”“出檐深远” → 体现唐代建筑的雄浑有力

# 塔

基本组成：塔刹、塔身、塔基（基座和地宫）

  • 楼阁式塔的节奏感和飘逸感
  • 密檐式塔的质朴 浑然一体 通常出檐很短
  • 覆钵式塔的有力。（北京白塔）
  • 过街塔很有意思，讲求 “与人方便”—— 我在其中看出了一种 “路德宗” 的意味，削减刻意的行为和仪式，因信称义。

演变 - 由于木制容易腐坏焚毁，只要技术能达到就演变为了砖石或者砖木。

  • 天宁寺 密檐，甚为高大的须弥座；莲瓣，烘托佛教氛围。强疏密对比，节奏感。收放秩序。

    （最近一定找时间去看看！！）

# 亭

一种前期还有些实用性，后来越来越偏向纯文化性的建筑。与审美结合极为密切。

文化特征为融于自然，一种有机建筑 （这个理念，颇为现代……）

  • 关注其 造型本身、空间的通透、造型与地形环境的协调
  • 激发内心独白的空间，往往有精神价值。

好棒的空间心理学案例…… 正因为其往往选址在能看到胜景的地方，人烟不能稠密，还有其空间的通透。那种 “既开敞又独立” 的平衡真实可贵。真是超凡脱俗的建筑。虚的特质。

  • 几种职能
    1. 自身作为构景的因素，增加活跃（远看）
    2. 人机工学的考虑，给人休息。
    3. 新的审美出发点，观景空间。

# 雕梁画栋 - 建筑构件

# 顶

为了保护土木结构，需要出挑；出挑多了又太沉重，于是逐渐形成反宇飞檐

  • 可以观察的维度，也就是体现房屋 “性格” 的特点：

    檐口是否 “软”- 飞檐上翘的幅度。脊饰的形态。梁思成评价清代的雕饰称极端程式化。几何形体的堆砌。

  • 雕饰 山墙中间挂下来的 “动物”- 通常是悬鱼 和鸱吻同理，一切为了防火。

# 架构

  • 举折 - 屋架高度，屋面坡度。

    大抵上越新的建筑坡度越陡峭，给人一种威严耸立的感觉。斗拱也变小，出檐收缩。

  • 推山 - 45 度方向看脊，不垂直而有曲线。

  • 雀替 - 柱上方与梁、枋交接的座。常作为鸟翼状。明清建筑外檐下常施雀替，可以观察一下。

# 柱

径高比体现性格。宋代起逐渐趋于细长。

侧脚 - 外檐柱微微向内倾斜，以达到垂直的视觉效果。一种视错觉。

内柱林立 - 内柱的密度影响室内空间给人的心理感受。太空旷会感到无所归依。恰到好处的立柱密度彰显向上的昂扬之势。

  • 与西方不同，未出现与人体有关的意象（多立克、爱奥尼、柯林斯）

# 斗拱

  • 实用承重构件，通过逐层挑出，分力

  • 文化符号。作者对其的赏析：“错综” 和 “理性” 的美感。复杂但是又有逻辑的东西，大家都喜欢吧。

  • 产生的根源：木材具有石材没有的韧性，但是刚度有限。就需要斗拱来分散屋檐承受的剪力。

    最初也是一个支柱（擎檐柱）发展来的。然后变成斜着的支撑。然后变成拱。有一点点飞扶拱的意思！一开始是一整面墙，后来变得凌空。

    从实向虚，从多到少。

# 门

  • 四合院。大门不设在中轴线上而通常在东南角，由于风水，东南是巽位，入。

    垂花门，最为精致的一道门。不是大门但在中轴线上，通常正对着厅堂。

# 窗

内外空间的分隔与延续、渗透。漏窗 这个说法很有趣 景物从窗户渗漏近来。

对建筑封闭性的否定。

窗的高贵在于视线通过，肢体不通过。眺望的工具。建筑呼吸的所在，空间气韵连通。

心理上，既受到庇护又不封闭。审美上，对大自然的选择性的 “剪裁”。

# 砖 & 瓦

  • 砖：泥土窑炼而成，不应与 “石” 混淆。

  • 瓦：瓦的横断面通常是弧形，一垄仰一垄俯，垄间相构。这样说，流水瓦应该在仰垄下方。

    青瓦 与 琉璃瓦 —— 彩色釉陶。

    脊饰：仙人指路 - 龙 - 凤 - 狮子 - 麒麟 - 天马 - 海马 - 鱼 - 獬 - 犼 - 猴。“防火大队” 哈哈哈有趣的比喻。

  • 瓦当：一个可以观察的方面！文字瓦当，自然瓦当，宗教瓦当。

# 栏杆

  • 栏板 + 望柱，仿木结构。二板一柱，形成节奏。

  • 古诗里大量表述，形成一种寄托思绪的意象：

    “把吴钩看了，栏杆拍遍，无人会，登临意。”“独自莫凭栏，无限江水，别时容易见时难。”

# 台基

  • 方形发展为须弥座 —— 对佛经挑选式的利用和吸收。佛教的须弥山和须弥座的三大特点：
    • 位于世界中心（中国人相信自己处于中心，对中心充满崇拜）
    • 坚固不坏（弥补土木结构容易朽坏的缺点）
    • 高耸无比（在观念和情感上实现现实中造不出的高度）

# 铺地

很有意思的说法：“句号”。

“人与自然的关系处于对立又统一的矛盾状态。人总愿意将他的一切文化方式带到自然的每一个角落，企望在一切领域打上人的文化烙印。…… 建筑作为人占有，改造自然的一种文化方式，同样要求全面实现人对自然的把握。…… 铺地作为最后一道工序，为改造画上一个完美的句号。从此人居住在一个自己建造的六合宇宙之中。“

  • 依照材料分类
    1. 石灰沙子鹅卵石 —— 比较随意的铺地。
    2. 砖铺地 / 砖铺墁
      • 长砖 - 长方形
      • 方砖 - 比长砖高级，或许是最规整吧
      • 金砖 - 皇家建筑。打磨之后，还要上一层蜡。
  • 场合 —— 室外
    • 散水 - 在建筑物较近的范围，保护地基不受雨水侵蚀。宽度根据出檐远近铺设，承接檐上流下的水。有的以石为材。
    • 甬铺 - 交通线上。常用方砖。
    • 园林环境中，往往更加随意。由于是最后一道工序，有时用建筑废料：碎砖碎瓦碎瓷片…

# 中西建筑比较

居然还有这个模块，也算惊喜。我一直都对各种 “沿革、比较、同化” 感兴趣，这一部分写的虽然简略，但也挺有启发性。

# 材料：土木和石头

西方：石材

  • 根源：采石技术；相信巨石的神性和巫性。巨石崇拜，在天地间用它来划分出一个安全的空间。

  • 气质上

    石材与人的关系更加紧张而具有推拒力，阳刚与力量，崇高与震慑。

    而木构建筑明显与人更加亲近，阴柔

这也决定了西方建筑的 “神性” 会更加强大？

# 美的逻辑：结构 & 雕塑

  • 中国建筑的 “三段式”，结构，逻辑清晰。

    • 林徽因：中国屋顶曲线只是结构直率自然的结果。并未经过矫揉造作。- 反宇飞檐的曲线美在重载结构通过自己的形态，在观感中失去沉重感，获得轻盈和欢愉。

    • 单体建筑，木结构逻辑通顺而严密。

    • 组合建筑，主次分明，主轴序列、副轴序列，轴线森列，层次清晰层层递进。 进深与院落 - 建筑结构的蒙太奇，非常严谨而理性。

      —— 这种建筑结构体现了社会结构。“家族之制” 的影响，等级观念和团组观念。也就是费孝通所言的差序格局

  • 西方建筑：雕塑美

    更加强调塑造而不是结构。如 “柱式” 基本与承重无关，只为了抽象表现人体的雕塑美。

    极为重视外立面的推敲。个体的体量与轮廓。

    • 现代主义建筑，如蓬皮杜中心，将结构直接外露，就是一种对于雕塑美的倾覆挑战，所以不容易被接受。

    建筑师往往都是雕塑家。

  • 关于建筑与人和环境

    中国人：天人合一。

    • 建筑是自然环境系统的有机构成。不是征服自然而是努力亲和融渗在自然中。柔顺，天然。
    • 将 “大” 通过群体组合来展现。

    欧洲人：努力突出个体特征，努力建的高大。

    • 个体崇高与个性。不具有 “礼” 的特性。

# 庭院和广场

  • 中国建筑的空间造型、生活情调和美学效果，通过群体组合来展现。人流交往与空间联系的媒介：门、庭院。

    “建筑物之间的空间” 比想象的更重要。“庭院深深深几许，杨柳堆烟，帘幕无重数。”

    内敛 宁静 封闭

    家庭的空间，排他性。

  • 西方 - 广场与城市一同成长。是交往的公共场所，城市 “涵虚” 的所在。

    袒露、开敞、流动的文化心态

    社会集中的中心，包容性。

  • 这样看起来，西方空间是 “大众” 而中国空间是 “私人”，似乎不太符合我们对此二者的印象，

    但是仔细想来广场的大众是独立个体组成的大众，中国庭院是实质为群体的私人。

    就好像，一个人去热闹的咖啡馆 和 拖家带口去深山老林。

    前者的人更多公众性更强，但受到来自 “人” 的羁绊无疑比后者更少，无疑流动性和自由性更强。

    不由得想起 b 站上看到一个社会学 up 主关于人际关系的解说，也借用 “差序格局” 的概念，我们的关系像涟漪一样的层级。

    强连接和弱连接 广场上极多的弱连接，庭院里少而永远稳固的强连接。

    这就好像我们自己社交的模式，也可以用这两个模型来分类。Sehr Interessant.

  • 好的，下一集预告：《广场》

# 神性与人性

  • 西方建筑中神性的体现更加重要。

  • 中国 “淡于宗教”（梁漱溟语）—— 尽可能收敛神的灵光而舒展人的姿态。很世俗，相当世俗，没有教权而只有皇权。

    证据是寺院建筑都不怎么虔诚，“渎神” 行为普遍。

    儒家所推崇的 “礼”（等级序列）与佛家的四大皆空诸法无常，其实非常矛盾。但是佛寺建筑只要允许，全部采用中轴线对称格局。佛寺还会出现彰显权势的龙纹，可谓不伦不类。

    佛塔原初是参拜之用，却变得可以登临，毫无神性。
